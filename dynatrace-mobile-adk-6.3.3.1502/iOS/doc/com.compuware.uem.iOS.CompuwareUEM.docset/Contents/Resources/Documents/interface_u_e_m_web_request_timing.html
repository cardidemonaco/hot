- (CPWR_StatusCode) startWebRequestTiming

Manaully start timing a web request.

The DynatraceUEM ADK automatically times web requests made using NSURLRequest, NSURLConnection, NSURLProtocol, NSString, UIWebView, and ASIHTTPRequest. If you use an alternate technology to make web requests and want to time them, use the getRequestTagHeader method, adding that information to your request, and then this method to start the timing.

Returns
Returns a CPWR_StatusCode
